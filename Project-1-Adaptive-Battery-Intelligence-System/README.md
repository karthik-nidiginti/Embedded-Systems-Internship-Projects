# Adaptive Battery Intelligence System

## Overview

This project is an ESP32-based Adaptive Multi-Cell Battery Intelligence Engine designed to monitor a simulated 4-cell lithium battery pack in real time.

The system reads analog voltages from four battery cells, calculates pack average voltage, imbalance percentage, strongest and weakest cells, and classifies battery health conditions.

## Features

* Real-time battery monitoring
* 4-cell voltage sensing
* Average pack voltage calculation
* Battery imbalance detection
* Strongest and weakest cell identification
* HEALTHY, MINOR, CRITICAL, and PACK FAILURE states
* LCD display monitoring
* Blynk IoT cloud monitoring
* Relay protection system
* Buzzer alert system

## Components Used

* ESP32
* LCD I2C Display
* Potentiometers
* LEDs
* Buzzer
* Relay Module
* Blynk IoT

## Software Used
